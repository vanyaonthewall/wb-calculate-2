# Логика расчётов — Калькулятор ремонта ПВЗ Wildberries v9

---

## Панель параметров

| Параметр | Переменная | Варианты |
|---|---|---|
| Общая площадь | `S` (м²) | 15–200, пресеты: 25 / 35 / 50 / 75 |
| Текущее состояние | `STATE` | `0` Косметика · `1` White Box · `2` Бетон |
| Свободная планировка | `PLAN` | `0` нет · `1` есть |
| Высота потолков | `K_height` | до 2.7 м → `1.0` · 3 м → `1.15` · от 3.5 м → `1.30` |

> Регион определяется на сайте автоматически → `K_region`

---

## Зонирование

```
S_client  = S × K_zone         // клиентская зона, K_zone = 0.65 по умолчанию
S_storage = S × (1 − K_zone)   // складская зона
```

Кнопка ✏️ в шапке сметы — изменить S и/или K_zone.
Отображение: `50 м² · 33 кл / 17 скл`

---

## Геометрия

```
P                = 4 × √S × 1.1               // периметр, м.п.
S_walls          = P × H                       // все стены, м²
S_walls_client   = S_walls × K_zone           // стены клиентской зоны
S_walls_storage  = S_walls × (1 − K_zone)     // стены склада
S_floor_client   = S × K_zone                 // пол клиентской зоны
S_floor_storage  = S × (1 − K_zone)           // пол склада
S_ceiling        = S                           // потолок
S_partition      = H × (√S × 0.5)             // перегородка (только PLAN=1)

N_fitting_rooms  = MAX(2, FLOOR(S / 25))       // примерочные
N_lights_client  = CEIL(S_client / 6)          // светильники клиент. зоны
N_lights_storage = CEIL(S_storage / 10)        // светильники склада
N_cameras        = MAX(3, CEIL(S / 20))        // камеры
```

## Коэффициенты

### Состояние помещения

| STATE | K_st (стены) | K_fl (пол) | K_cl (потолок) |
|---|---|---|---|
| `0` Косметика | 0.4 | 0.0 | 0.3 |
| `1` White Box | 0.7 | 1.0 | 0.7 |
| `2` Бетон | 1.0 | 1.0 | 1.0 |

> STATE=0 + выбрана штукатурка вручную → K_st = 0.9

### Высота потолков

| Выбор | H ср. (м) | K_height |
|---|---|---|
| До 2.7 м | 2.5 | 1.0 |
| 3 м | 2.85 | 1.15 |
| От 3.5 м | 3.25 | 1.30 |

### Региональный коэффициент

| Регион | K_region |
|---|---|
| Москва / МО | 1.0 |
| СПб / Ленобласть | 0.90 |
| Города-миллионники | 0.75 |
| Остальные | 0.60 |

## Формулы расчёта

### Категория 1 — Внутренняя отделка

```
// Стены — клиентская зона
COST_walls_client  = S_walls_client × K_st × K_height
                   × (C_mat_walls_client + C_work_walls) × K_region

// Стены — складская зона (упрощённая подготовка)
COST_walls_storage = S_walls_storage × K_st × K_height
                   × (C_mat_walls_storage + C_work_walls_storage) × K_region

// Пол — клиентская зона (активен один из двух)
COST_floor_tile    = S_floor_client × K_fl × (2 100 + 2 300) × K_region  // плитка (дефолт)
COST_floor_lino_c  = S_floor_client × K_fl × (780 + 575) × K_region      // линолеум (тогл)

// Пол — складская зона (всегда линолеум)
COST_floor_storage = S_floor_storage × K_fl × (780 + 575) × K_region

// Стяжка (только STATE=2, добавляется к полу)
COST_screed        = S × (320 × 1.5 + 1 100) × K_region

// Потолок (активен один из двух)
COST_grillato      = S_ceiling × (1 800 + 1 500) × K_region               // грильято (дефолт)
COST_ceiling_paint = S_ceiling × K_cl × K_height × (80 + 575) × K_region // покраска (тогл)

// Перегородка (только PLAN=1)
COST_partition     = S_partition × (1 500 + 2 850) × K_region

COST_1 = COST_walls_client
       + COST_walls_storage
       + COST_floor_client   // tile или lino_c
       + COST_floor_storage
       + COST_screed          // 0 если STATE ≠ 2
       + COST_ceiling          // grillato или paint
       + COST_partition        // 0 если PLAN=0
```

**Базовые ставки (Москва, ₽/м²):**

| Статья | Материал | Работа |
|---|---|---|
| Покраска стен (клиент) | ~150 | 600–800 |
| +Штукатурка | +155 | +750–1 400 |
| +Шпаклёвка | +72 | +400–700 |
| Покраска стен (склад) | ~80 | 350–600 |
| Плитка (клиент) | 2 100 | 2 200 |
| Линолеум | 780 | 575 |
| Грильято | 1 800 | 1 500 |
| Покраска потолка | 80 | 575 |
| Перегородка ГКЛ | 1 500 | 2 850 |

### Категория 2 — Электрика и свет

```
// Прокладка сети (STATE≥1) или только розетки (STATE=0)
COST_elec          = S × (STATE=0 ? 1 200 : 3 500) × K_region

// Светильники — клиентская зона
COST_lights_client = N_lights_client × (3 000 + 800) × K_region

// Аварийное освещение
COST_emergency     = CEIL(S / 50) × (1 850 + 650) × K_region

// Светильники — склад
COST_lights_stor   = N_lights_storage × (1 150 + 500) × K_region

// Климат (если тоглы включены)
COST_zavesa        = 1 × (10 000 + 4 000) × K_region   // завеса: матер. + монтаж
COST_ac            = 1 × (35 000 + 9 000) × K_region   // кондиционер: матер. + монтаж

COST_2 = COST_elec
       + COST_lights_client
       + COST_emergency
       + COST_lights_stor
       + COST_zavesa   // 0 если тогл выключен
       + COST_ac       // 0 если тогл выключен
```

// Видеонаблюдение

// Пожарная безопасность

COST_it = [55 000 × вкл_ноутбук]
        + [3 500 × вкл_сканер]
        + [3 500 × вкл_роутер]

COST_3 = COST_cameras + COST_nvr + COST_fire + COST_it
