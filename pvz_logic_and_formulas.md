# Логика расчётов — Калькулятор ремонта ПВЗ Wildberries

---

## Экран 1 — Быстрый расчёт

### Входные данные

| Параметр | Переменная | Варианты |
|---|---|---|
| Общая площадь | `S` (м²) | 15–200, пресеты: 25 / 35 / 50 / 75 |
| Текущее состояние | `STATE` | `0` Косметика · `1` White Box · `2` Бетон |
| Свободная планировка | `PLAN` | `0` нет · `1` есть (нужна перегородка) |

### Дефолтные значения (применяются пока не уточнено на Экране 2)

| Параметр | Переменная | Дефолт |
|---|---|---|
| Высота потолков | `K_height` | `1.15` (H = 2.85 м) |
| Электрика | `ELEC` | `1` (монтаж с нуля) |
| Стены | `WALLS` | STATE=0 → `0`; STATE≥1 → `1` |
| Санузел | `TOILET` | `0` (нет) |
| Пол клиентской зоны | `FLOOR_client` | плитка |
| Потолок | `CEILING` | грильято |
| Регион | `K_region` | определяется сайтом автоматически |

### Результат

Отображается как `~X ₽` — приблизительная стоимость.
Обновляется в реальном времени при изменении S, STATE, PLAN.

## Экран 2 — Уточнение деталей

### Входные данные (дополнительно к Экрану 1)

| Параметр | Переменная | Варианты |
|---|---|---|
| Высота потолков | `H` → `K_height` | до 2.7 м → `1.0` · 3 м → `1.15` · от 3.5 м → `1.30` |
| Электрика | `ELEC` | `0` Обновить свет · `1` Монтаж с нуля |
| Стены | `WALLS` | `0` Только покраска · `1` Выровнять и покрасить |
| Санузел | `TOILET` | `0` Нет · `1` Только санфаянс · `2` Прокладка труб |

> Регион не отображается — определяется автоматически на сайте.

## Экран 3 — Смета

### Шапка сметы

```
S м²  ·  S_client кл / S_storage скл
```

- `S_client = S × 0.65`
- `S_storage = S × 0.35`
- K_zone = 0.65 по умолчанию (брендбук WB)
- Кнопка «карандаш» — изменить S и/или K_zone

---

## Геометрические переменные

```
P             = 4 × √S × 1.1            // периметр (м.п.)
S_walls       = P × H                   // площадь всех стен (м²)
S_walls_client  = S_walls × K_zone      // стены клиентской зоны
S_walls_storage = S_walls × (1−K_zone)  // стены склада
S_floor_client  = S × K_zone            // пол клиентской зоны
S_floor_storage = S × (1−K_zone)        // пол склада
S_ceiling     = S                        // потолок
S_partition   = H × (√S × 0.5)         // перегородка (только PLAN=1)

N_fitting_rooms = MAX(2, FLOOR(S / 25))  // примерочные
N_lights_client = CEIL(S_client / 6)     // светильники клиент. зоны
N_lights_storage = CEIL(S_storage / 10) // светильники склада
N_cameras     = MAX(3, CEIL(S / 20))    // камеры
```

## Коэффициенты

### Состояние помещения

| STATE | K_st (стены) | K_fl (пол) | K_cl (потолок) |
|---|---|---|---|
| `0` Косметика | 0.4 | 0.0 | 0.3 |
| `1` White Box | 0.7 | 1.0 | 0.7 |
| `2` Бетон | 1.0 | 1.0 | 1.0 |

> STATE=0 + WALLS=1 → K_st принудительно = `0.9`

### Высота потолков

| Выбор | H ср. | K_height |
|---|---|---|
| До 2.7 м | 2.5 | 1.0 |
| 3 м | 2.85 | 1.15 |
| От 3.5 м | 3.25 | 1.30 |

### Региональный коэффициент

| Регион | K_region |
|---|---|
| Москва / МО | 1.0 |
| СПб / Ленобласть | 0.90 |
| Города-миллионники | 0.75 |
| Остальные города | 0.60 |

## Формулы по категориям

### Категория 1 — Внутренняя отделка

```
// Стены — клиентская зона
COST_walls_client = S_walls_client × K_st × K_height × (C_mat_walls_client + C_work_walls) × K_region

// Стены — складская зона (упрощённая подготовка)
COST_walls_storage = S_walls_storage × K_st × K_height × (C_mat_walls_storage + C_work_walls_storage) × K_region

// Пол — клиентская зона (плитка — дефолт)
COST_floor_client_tile = S_floor_client × K_fl × (2 100 + 2 300) × K_region

// Пол — клиентская зона (линолеум — тогл)
COST_floor_client_lino = S_floor_client × K_fl × (780 + 575) × K_region

// Пол — складская зона (линолеум — всегда)
COST_floor_storage = S_floor_storage × K_fl × (780 + 575) × K_region

// Потолок — грильято (дефолт)
COST_ceiling_grillato = S_ceiling × (1 800 + 1 500) × K_region

// Потолок — покраска (тогл, альтернатива грильято)
COST_ceiling_paint = S_ceiling × K_cl × K_height × (80 + 575) × K_region

// Перегородка (только PLAN=1)
COST_partition = S_partition × (1 500 + 2 850) × K_region

// Итог категории 1
COST_1 = COST_walls_client
       + COST_walls_storage
       + COST_floor_client       // tile или lino — зависит от тогла
       + COST_floor_storage
       + COST_ceiling             // grillato или paint — зависит от тогла
       + COST_partition           // 0 если PLAN=0
```

**Базовые значения стоимости (Москва, ₽/м²):**

| Статья | Материал | Работа |
|---|---|---|
| Стены клиент. (покраска) | ~150 | 600–800 |
| Стены клиент. (+штукатурка, WALLS=1) | +150 | +1 300–2 100 |
| Стены склад (покраска) | ~80 | 350–600 |
| Пол плитка (клиент) | 2 100 | 2 200 |
| Пол линолеум | 780 | 570 |
| Потолок грильято | 1 800 | 1 500 |
| Потолок покраска | 80 | 575 |
| Перегородка ГКЛ | 1 500 | 2 850 |

---

### Категория 2 — Электрика и свет

```
// Электромонтаж
COST_elec = S × (ELEC=0 ? 1 200 : 3 500) × K_region

// Светильники — клиентская зона
COST_lights_client = N_lights_client × (C_light_mat + C_light_work) × K_region
// C_light_mat = 2 200–3 500 ₽/шт., C_light_work = 600–1 000 ₽/шт.

// Аварийное освещение (обязательно)
COST_emergency = CEIL(S / 50) × (1 850 + 650) × K_region

// Светильники — склад
COST_lights_storage = N_lights_storage × (1 150 + 500) × K_region

// Итог категории 2
COST_2 = COST_elec + COST_lights_client + COST_emergency + COST_lights_storage
```

---

### Категория 3 — Безопасность и IT

```
// Видеонаблюдение
COST_cameras = N_cameras × (3 750 + 2 250) × K_region
COST_nvr     = (9 000 + 4 000) × K_region   // NVR + HDD + монтаж

// Пожарная безопасность
COST_fire = (11 500 + 1 750 + 10 000) × K_region

// IT-оборудование (без K_region, фиксированные цены)
COST_it = [ноутбук × вкл] + [сканер × вкл] + [роутер × вкл]
// ноутбук: 40 000–70 000 ₽; сканер: 2 000–5 000 ₽; роутер: 2 000–5 000 ₽

// Итог категории 3
COST_3 = COST_cameras + COST_nvr + COST_fire + COST_it
```

---

### Категория 4 — Санузел

```
COST_4 = TOILET=0 ? 0
       : TOILET=1 ? 45 000 × K_region
       : TOILET=2 ? 120 000 × K_region
```

> Суммы фиксированные — ориентировочные. Уточняются после обследования.

---

### Категория 5 — Мебель

```
// Рассчитывается автоматически по площади, пользователь корректирует кол-во в смете
// Без K_region — фиксированные цены поставщика WB

COST_5 = Σ (кол-во × цена) по всем позициям мебели
```

**Формулы количества:**

| Позиция | Формула |
|---|---|
| Примерочные | `MAX(2, FLOOR(S / 25))` |
| Шторы в примерочную | `= кол-ву примерочных` |
| Скамейка белая | `2` (всегда) |
| Скамейка розовая | `1 + FLOOR((S − 50) / 30)` при S > 50, иначе `1` |
| Зеркало в зал | `1` (всегда) |
| Урна | `1` (всегда) |
| Стол выдачи 2 яч. | `MAX(1, FLOOR(S / 40))` при S < 80, иначе заменяется |
| Стол выдачи 3 яч. | `1` если S > 80 |
| Стол проверки маленький | `1` (всегда) |
| Стол проверки большой | `1` если S > 70 |
| Стол менеджера | `1` (всегда) |
| Стеллаж складской | `CEIL(S_storage / 15)` |
| Стенд, режимник | `1` (всегда) |

---

### Категория 6 — Резерв 12%

```
COST_before_reserve = COST_1 + COST_2 + COST_3 + COST_4 + COST_5
COST_reserve        = COST_before_reserve × 0.12
COST_TOTAL          = COST_before_reserve + COST_reserve
```
