## Панель параметров — входные данные

| Параметр | Переменная | Варианты |
|---|---|---|
| Общая площадь | `S` (м²) | 15–200, пресеты: 25 / 35 / 50 / 75 |
| Текущее состояние | `STATE` | `0` Косметика · `1` White Box · `2` Бетон |
| Свободная планировка | `PLAN` | `0` нет · `1` есть |
| Высота потолков | `H` → `K_height` | до 2.7 м → `1.0` · 3 м → `1.15` · от 3.5 м → `1.30` |

> Регион определяется автоматически на сайте → `K_region`

---

## Зонирование (автоматически из S)

```
S_client  = S × 0.65   // клиентская зона
S_storage = S × 0.35   // складская зона
```

Кнопка ✏️ в шапке сметы — изменить S и/или K_zone вручную.
Отображение в шапке: `35 м² · 23 кл / 12 скл`

---

## Геометрия (вычисляется автоматически)

```
P                = 4 × √S × 1.1
S_walls          = P × H
S_walls_client   = S_walls × K_zone
S_walls_storage  = S_walls × (1 − K_zone)
S_floor_client   = S × K_zone
S_floor_storage  = S × (1 − K_zone)
S_ceiling        = S
S_partition      = H × (√S × 0.5)        // только PLAN=1

N_fitting_rooms  = MAX(2, FLOOR(S / 25))
N_lights_client  = CEIL(S_client / 6)
N_lights_storage = CEIL(S_storage / 10)
N_cameras        = MAX(3, CEIL(S / 20))
```

## Коэффициенты

### Состояние помещения

| STATE | K_st (стены) | K_fl (пол) | K_cl (потолок) |

| `0` Косметика | 0.4 | 0.0 | 0.3 |

### Высота потолков

| Выбор | K_height |
|---|---|
| До 2.7 м | 1.0 |

### Региональный коэффициент

| Регион | K_region |
|---|---|
| Москва / МО | 1.0 |
| СПб / Ленобласть | 0.90 |
| Города-миллионники | 0.75 |
| Остальные | 0.60 |

## Тогглы сметы — полная таблица

Каждая строка сметы — отдельный тогл. Параметры панели автоматически включают / выключают нужные тогглы при изменении.

### Категория 1 — Внутренняя отделка

| Тогл / работа | Дефолт | Включается при | Выключается при | Взаимоисключение |
|---|---|---|---|---|
| Демонтаж старого покрытия | ВЫКЛ | STATE=0, вручную | — | — |
| Штукатурка стен | ВЫКЛ | STATE=2 | STATE=0 | — |
| Шпаклёвка стен финишная | ВКЛ | STATE≥1 | STATE=0 | — |
| Грунтовка стен | ВКЛ | всегда | — | — |
| Покраска стен (клиент) | ВКЛ | всегда | — | — |
| Покраска акцентной стены | ВКЛ | всегда | — | — |
| Покраска стен (склад) | ВКЛ | всегда | — | — |
| Покраска радиаторов | ВКЛ | всегда | — | — |
| Бренд-стена (обои WB) | ВКЛ | всегда | — | — |
| Стяжка пола | ВЫКЛ | STATE=2 | STATE≠2 | — |
| Укладка плитки (клиент) | ВКЛ | K_fl > 0 | STATE=0 | ↔ Линолеум клиент |
| Линолеум (клиент) | ВЫКЛ | вручную | — | ↔ Плитка клиент |
| Укладка линолеума (склад) | ВКЛ | всегда | — | — |
| Монтаж потолка грильято | ВКЛ | всегда | — | ↔ Покраска потолка |
| Покраска потолка | ВЫКЛ | вручную | — | ↔ Грильято |
| Установка перегородки ГКЛ | ВЫКЛ | PLAN=1 | PLAN=0 | — |

### Категория 2 — Электрика и свет

| Тогл / работа | Дефолт | Включается при | Выключается при | Взаимоисключение |
|---|---|---|---|---|
| Монтаж электрики с нуля | ВКЛ | STATE≥1 | STATE=0 | ↔ Обновление электрики |
| Обновление электрики | ВКЛ | STATE=0 | STATE≥1 | ↔ Монтаж с нуля |
| Монтаж светильников (клиент) | ВКЛ | всегда | — | — |
| Аварийное освещение | ВКЛ | всегда | — | — |
| Монтаж светильников (склад) | ВКЛ | всегда | — | — |
| Тепловая завеса | ВЫКЛ | вручную | — | — |
| Кондиционер | ВЫКЛ | вручную | — | — |

### Категория 3 — Безопасность и IT

| Тогл / работа | Дефолт | Включается при | Выключается при |
|---|---|---|---|
| Монтаж камер видеонаблюдения | ВКЛ | всегда | — |
| Монтаж видеорегистратора | ВКЛ | всегда | — |
| Пожарная сигнализация | ВКЛ | всегда | — |
| Ноутбук для сотрудника | ВКЛ | всегда | вручную |
| Сканер штрих-кодов | ВКЛ | всегда | вручную |
| Роутер / Wi-Fi | ВКЛ | всегда | вручную |

### Категория 4 — Санузел

| Тогл / работа | Дефолт | Включается при |
|---|---|---|
| Монтаж санфаянса (к готовым выводам) | ВЫКЛ | вручную → TOILET=1 |
| Прокладка труб + санузел под ключ | ВЫКЛ | вручную → TOILET=2 |

> Санузел — единственная категория, которую пользователь включает вручную с нуля. По умолчанию всё выключено.

### Категория 5 — Мебель

| Тогл / позиция | Дефолт | Кол-во |
|---|---|---|
| Примерочная | ВКЛ | `MAX(2, FLOOR(S/25))` |
| Штора в примерочную | ВКЛ | = примерочным |
| Скамейка белая | ВКЛ | 2 |
| Скамейка розовая | ВКЛ | `1 + FLOOR(MAX(0, S−50)/30)` |
| Зеркало в зал | ВКЛ | 1 |
| Урна | ВКЛ | 1 |
| Стол выдачи 2 яч. | ВКЛ | `MAX(1, FLOOR(S/40))` при S≤80 |
| Стол выдачи 3 яч. | ВКЛ если S>80 | 1 |
| Стол проверки маленький | ВКЛ | 1 |
| Стол проверки большой | ВКЛ если S>70 | 1 |
| Стол менеджера | ВКЛ | 1 |
| Стеллаж складской | ВКЛ | `CEIL(S_storage/15)` |
| Стенд потребителя | ВКЛ | 1 |
| Режимник WB | ВКЛ | 1 |

## Формулы расчёта

### Категория 1 — Внутренняя отделка

```
// Стены
COST_walls_client   = S_walls_client × K_st × K_height × (C_mat_walls_client + C_work_walls) × K_region
COST_walls_storage  = S_walls_storage × K_st × K_height × (C_mat_walls_storage + C_work_walls_storage) × K_region

// Пол — клиентская зона (один из двух активен)
COST_floor_tile  = S_floor_client × K_fl × (2 100 + 2 300) × K_region   // плитка (дефолт)
COST_floor_lino  = S_floor_client × K_fl × (780 + 575) × K_region        // линолеум (тогл)

// Пол — склад (всегда линолеум)
COST_floor_storage = S_floor_storage × K_fl × (780 + 575) × K_region

// Потолок (один из двух активен)
COST_ceiling_grillato = S_ceiling × (1 800 + 1 500) × K_region           // грильято (дефолт)
COST_ceiling_paint    = S_ceiling × K_cl × K_height × (80 + 575) × K_region // покраска (тогл)

// Стяжка (только STATE=2)
COST_screed = S × (320 × 1.5 + 1 100) × K_region

// Перегородка (только PLAN=1)
COST_partition = S_partition × (1 500 + 2 850) × K_region

COST_1 = COST_walls_client + COST_walls_storage
       + COST_floor_client  // tile или lino
       + COST_floor_storage
       + COST_ceiling        // grillato или paint
       + COST_screed         // 0 если STATE≠2
       + COST_partition      // 0 если PLAN=0
```

### Категория 2 — Электрика и свет

```
COST_elec            = S × (STATE=0 ? 1 200 : 3 500) × K_region
COST_lights_client   = N_lights_client × (3 000 + 800) × K_region
COST_lights_storage  = N_lights_storage × (1 150 + 500) × K_region
COST_emergency       = CEIL(S/50) × (1 850 + 650) × K_region

COST_2 = COST_elec + COST_lights_client + COST_lights_storage + COST_emergency
       + [тепловая завеса: 9 000 × K_region если вкл]
       + [кондиционер: 35 000 × K_region если вкл]
```

### Категория 3 — Безопасность и IT

```
COST_cameras = N_cameras × (3 750 + 2 250) × K_region
COST_nvr     = (9 000 + 4 000) × K_region
COST_fire    = (11 500 + 1 750 + 10 000) × K_region
COST_it      = [ноутбук × вкл] + [сканер × вкл] + [роутер × вкл]
// без K_region

COST_3 = COST_cameras + COST_nvr + COST_fire + COST_it
```

### Категория 4 — Санузел

```
COST_4 = TOILET=0 ? 0
       : TOILET=1 ? 45 000 × K_region
       : TOILET=2 ? 120 000 × K_region
```

### Категория 5 — Мебель

```
COST_5 = Σ (кол-во × цена) // без K_region
```

### Итог

```
COST_before_reserve = COST_1 + COST_2 + COST_3 + COST_4 + COST_5
COST_reserve        = COST_before_reserve × 0.12
COST_TOTAL          = COST_before_reserve + COST_reserve
```

## Логика AUTO-переключения тогглов при смене STATE

При изменении STATE на панели — автоматически меняются дефолты следующих тогглов:

| Тогл | STATE=0 | STATE=1 | STATE=2 |
|---|---|---|---|
| Демонтаж покрытия | ВЫКЛ | ВЫКЛ | ВЫКЛ |
| Штукатурка стен | ВЫКЛ | ВЫКЛ | ВКЛ |
| Шпаклёвка финишная | ВЫКЛ | ВКЛ | ВКЛ |
| Стяжка пола | ВЫКЛ | ВЫКЛ | ВКЛ |
| Укладка плитки (клиент) | ВЫКЛ | ВКЛ | ВКЛ |
| Монтаж с нуля (электрика) | ВЫКЛ | ВКЛ | ВКЛ |
| Обновление электрики | ВКЛ | ВЫКЛ | ВЫКЛ |

При изменении PLAN:

| Тогл | PLAN=0 | PLAN=1 |
|---|---|---|
| Установка перегородки ГКЛ | ВЫКЛ | ВКЛ |
